#Stream and Socket.io

## At first

```
$ npm install for all deps
```
## After that you can choose:

### Stream:
#### 1 Stream -> from txt to txt.gz
```
$ cd stream
$ node zipper.js
```
#### 2 Stream -> from txt.gz to txt
```
$ cd stream
$ node unzipper.js
```
###Socket.io:

```
$ cd socket
$ npm start
$ open localhost:3000 in browser
```
